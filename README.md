# MoviesNow

ფინალური პოექტი ვებ-პროგრამირების საგანში

## Info

- **სათაური**: MoviesNow
- **აღწერა**: საიტი იქნება განკუთვნილი ფილმების მოსაძებნად და საყურებლად
- **API**:  [api.adjaranet.com](api.adjaranet.com)

## Features

1) - [ ] მთავარი გვერდი
     - [ ] ძებნა ფილმის სათაურით
     - [ ] ძებნის ფილტრი
         - [ ] წლების ფილტრი
         - [ ] ჟანრის ფილტრი
         - [ ] ქვეყნის ფილტრი

2) - [ ] ფილმის გვერდი
      - [ ] ინფორმაცია ფილმეზე
      - [ ] ფილმის ყურება
      - [ ] მსახიობები ფილმში

3) - [ ] მასახიობის გვერდი
      - [ ] ინფორმაცია მსახიობზე
      - [ ] მსახიობის ფილმები
